# AI Agent (Python + OpenAI)

A lightweight AI agent built with **Python** that connects to the **OpenAI API** and runs through a local API server.  
The agent processes prompts and generates responses using OpenAI models.

---

## Features

- AI responses powered by OpenAI
- Simple Python backend
- API server using Uvicorn
- Easy to extend with tools (web search, database, etc.)
- Optional offline mode support

---

## Tech Stack

- Python 3.10+
- FastAPI
- OpenAI API
- Uvicorn
